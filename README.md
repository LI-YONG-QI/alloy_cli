# Alloy CLI Demo

build by:

- [alloy](https://crates.io/crates/alloy)
- [clap](https://crates.io/crates/clap)

## Features

Available commands:

- `block`: get block number
- `balance`: get balance of address

## How to start ?

- Check all commands

```bash
cargo run -- --help
```

- Get block number

```bash
cargo run -- block
```

- Get balance

```bash
cargo run -- balance <YOUR ADDRESS e.g. 0x056703bb4E0866909E1767D9b079237D1C44962f>
```
